## JsonParse 组件

| 属性     | 说明                   | 类型                                                   | 默认值 |
| -------- | ---------------------- | ------------------------------------------------------ | ------ |
| jsonData | 需要传入 json 的行解析 | `object[]`                                             | -      |
| onSelect | 选择回调               | `(value: []) => { console.log('select item', value) }` | -      |
